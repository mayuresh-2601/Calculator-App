# 🧮 Calculator Web App

A simple and responsive Calculator web application built using **HTML**, **CSS**, and **JavaScript**.
The application performs basic arithmetic operations with a clean and user-friendly interface.

---

## 🚀 Live Demo

👉 https://mayuresh-2601.github.io/Calculator-App/

---

## 📌 Features

* Perform basic arithmetic operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Clear display functionality
* Simple and responsive user interface
* Real-time calculation results

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
